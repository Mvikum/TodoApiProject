# TodoApiProject

A simple ASP.NET Core Web API project for managing Todo items. This project demonstrates CRUD (Create, Read, Update, Delete) operations using a RESTful API.

## Features

- List all Todo items
- Create a new Todo item
- Update an existing Todo item
- Delete a Todo item

## Technologies Used

- ASP.NET Core
- C#
